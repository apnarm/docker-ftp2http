# ftp2http Daemon #

### Running docker ###

This service is run and deployed using [Docker](https://www.docker.com/).
The base image is [gliderlabs/alpine:3.2](https://github.com/gliderlabs/docker-alpine)

``` sourceCode
docker-compose build

# Run in background
docker-compose start

# Stop the services
docker-compose stop

OR

# Run in foreground
docker-compose up


```
